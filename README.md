# 🛒 E-commerce Full Stack QA Portfolio

This repository contains a comprehensive Quality Assurance (QA) project for an E-commerce platform, covering the entire testing lifecycle from manual strategy to automated execution.

---

## 🚀 Project Components

### 1. 📝 Manual Testing
- **Location:** `/manual_testing.md`
- **Focus:** User flows, registration, product search, and cart functionality.
- **Techniques:** Positive/Negative testing and Boundary Value Analysis (BVA).

### 2. 🔌 API Testing
- **Location:** `/api_testing/api_testing.json`
- **Tool:** Postman
- **Focus:** Validating RESTful API endpoints, status codes (200, 201), and JSON response structures.

### 3. 🤖 UI Automation
- **Location:** `/automation/ecommerce_automation.py`
- **Tech Stack:** Python + Selenium WebDriver
- **Focus:** Automated regression testing for critical user paths (Search & Cart validation).

---

## 🛠 Tech Stack & Tools
- **Automation:** Selenium WebDriver
- **Scripting:** Python
- **API Tools:** Postman
- **Documentation:** Markdown
- **Version Control:** Git & GitHub

---

## ⚙️ How to View/Run
- **Manual Cases:** View the `manual_testing.md` file directly in the browser.
- **API:** Import the `.json` file into Postman to see the request structures.
- **Automation:** 1. Ensure Python and Selenium are installed.
  2. Run: `python automation/ecommerce_automation.py`
